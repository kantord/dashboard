# How to use area chart
Here's an example code regarding the use of area chart: 

```yaml
dashboard "Example": 
  - 
    area chart: 
      columns: 
        - 
          - "Apples"
          - 30
          - 29
          - 25
          - 26
          - 27
          - 10
        - 
          - "Oranges"
          - 20
          - 21
          - 22
          - 20
          - 27
          - 19

```
The code above will render a area chart that looks like this:
