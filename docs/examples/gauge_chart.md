# How to use gauge chart
Here's an example code regarding the use of gauge chart: 

```yaml
---
  dashboard "Example": 
    - 
      gauge chart: 
        columns: 
          - 
            - "data"
            - 30

```
The code above will render a gauge chart that looks like this:
